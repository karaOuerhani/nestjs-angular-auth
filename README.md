# nestjs-angular-auth
Complete user authentication system with email verification built with NestJS backend and Angular frontend.

## Features
- ✅ User registration with email validation
- ✅ Secure login with JWT authentication
- ✅ Email verification with verification codes
- ✅ Modern responsive UI with PrimeNG components
- ✅ Modular and scalable architecture following best practices

## Technologies Used
- **Backend**: NestJS, Prisma ORM, PostgreSQL, JWT, bcrypt, Nodemailer
- **Frontend**: Angular 18, Reactive Forms, RxJS, PrimeNG, SCSS
- **Email Service**: Gmail SMTP integration
- **Database**: PostgreSQL with Prisma migrations

## Getting Started
1. Clone the repository
2. Install backend dependencies: `cd backend && npm install`
3. Install frontend dependencies: `cd frontend && npm install`
4. Set up PostgreSQL database and configure environment variables
5. Run backend: `npm run start:dev`
6. Run frontend: `ng serve`
